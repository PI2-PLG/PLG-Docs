Sobre o Produto

Dispositivo para Alertas de Incêndio no Cerrado

O Projeto Lobo-Guará projeto tem como principal motivação a urgência em, a partir dos dados coletados, detectar focos de incêndio no cerrado próximo a centros urbanos como também próximo a plantações. Assim permitindo que as medidas necessárias para a contenção dos focos de incêndio sejam tomadas no menor período possível. Tudo isso é possível através de um sistema de sensores que coletam informações do ambiente no momento e tudo isso é passado para um aplicativo que trata as informações.

O módulo PLG tem o formato de uma caixa média onde estão dispostos os sensores eletrônicos para coleta de dados do ambiente, além de outros componentes eletrônicos para seu funcionamento. Uma bateria e placa solar para que o monitoramento dos dados seja coletado de 30 em 30 minutos.

Funcionalidades

Notificações

O usuário recebe notificações em tempo real pelo app PLG quando um foco de incêndio é identificado, assim como notificações que dizem se o módulo está funcionando corretamente ou não.

Gráficos

O usuário pode visualizar os dados coletados por meio de gráficos que apresentam as taxas de umidade, temperatura e outros ao longo do dia. Assim como uma comparação entre esse dados.

Cadastro do Usuário

Ao se cadastrar o usuário tem o controle dos módulos na tela do aplicativo.

Monitoramento

O usuário pode visualizar as informações em tempo real de um módulo em específico, ou seja, o monitoramento de cada módulo.

Conheça nosso aplicativo!

O app PLG recebe os dados coletados pelos sensores do módulo e trata eles visualmente. Mostrando gráficos de monitoramento e comparação entre as taxas coletadas. Além de notificações que o usuário recebe quando um foco de incêndio é decctado.
